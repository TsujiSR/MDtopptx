# md2pptx-builder サンプルプレゼンテーション

このスライドはテスト用サンプルです。

## 機能一覧

- Markdownからスライド生成
- 背景画像とロゴの適用
- テンプレートの継承

---

# スライド機能の紹介

## 見出しのサポート

Markdownの見出し記法をサポートしています。

- `#` = スライドタイトル (H1)
- `##` = サブタイトル (H2)
- `###` = 小見出し (H3)

---

# フォーマットのサポート

テキストは**太字**や*斜体*もサポート。

## コードブロック

```python
def hello():
    print("Hello, World!")
    return True
```

---

# スライド区切りについて

スライドの区切りは次のいずれかを使用:

1. `---` (ハイフン3つ)
2. `<!-- pagebreak -->` (HTMLコメント)

どちらを使っても同じ結果になります。
